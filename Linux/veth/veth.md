# 虚拟网络设备Veth #

-> 参考：https://segmentfault.com/a/1190000009251098?utm_source=tag-newest

-> veth设备的特点
	-> veth和其他的网络设备都一样，一段连接的是内核协议栈
	-> veth设备是成对出现额，另一端2个设备彼此相连
	-> 一个设备收到协议栈的数据发送请求后，会将数据发送到另一个设备上去。
